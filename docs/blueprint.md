# **App Name**: Easy Bites

## Core Features:

- Intelligent Savoring Tool: A generative AI tool that recommends the perfect meal based on current weather, time of day, and user mood patterns.
- Live Gastronomy Catalog: Dynamic menu browsing with high-quality imagery, smooth category transitions, and rapid search indexing via Firestore.
- Frictionless checkout Flow: A refined 3-step checkout featuring slide-in cart dynamics, quantity management, and delivery threshold calculations.
- Unified Payment Hub: Integrated Razorpay, UPI QR visualization, and COD options for a versatile transaction experience.
- Culinary Progress Tracker: Real-time order state updates synchronized with the cloud to follow the meal from preparation to the doorstep.
- Command Center Dashboard: A streamlined admin interface for inventory control, menu toggling, and daily order analysis powered by Firestore.
- Precision Maps & Social: One-tap WhatsApp support and integrated Google Maps navigation for localized store discovery.

## Style Guidelines:

- Primary: Vibrant Crimson (#DC2626) to evoke energy and appetite. Background: Soft Pearl (#FAFAFA) for a clean, premium light-mode foundation. Accent: Rose-toned Crimson (#DC2664) used sparingly for high-interest UI elements like favorite icons and ratings.
- A sophisticated pairing of 'Space Grotesk' for modern, tech-forward headlines and 'Inter' for exceptionally clear, legible body text across all device sizes.
- Feather-weight outline icons with softened terminals, prioritizing common restaurant metaphors for intuitive navigation.
- A fluid, mobile-first design utilizing spacious cards with soft 16px border-radii and subtle elevations to create depth without clutter.
- Physics-based slide-in animations for the cart drawer and layout-shifts using Framer Motion to provide immediate tactile feedback.
